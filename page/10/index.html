Leetcode 141 Linked List Cycle(环形链表)

Posted on 2019-12-17 Edited on 2020-03-05 In leetcode

Difficulty : Easy

Type: Linked List && Two Pointers

Language : Java

  1. Clarification

    Given a linked list, determine if it has a cycle in it.

  2. Possible Solutions

    • HashSet
      Iterate all the nodes and use HashSet to store references of nodes. If head is not null, determine whether HashSet contain head. If not, add head into Set. After each interation, head = head.next
    • Faster and Slower Pointers
      In order to determine whether a linked list contains circle, we can set two pointers. One is slow pointer that only moves a step each time and another is fast pointer that moves two steps each time. If the list has circle, fast pointer will equal to slow pointer eventually.

  3. Code

HashSet’s runtime is slower than Fast and Slow Pointers because new a hashset will cost extra time. It doesn’t mean that this algorithm’s speed is lower than Fast and Slow Pointers.

  1. Test Cases

    Input : head = [3,2,0,-4], pos = 1

    Output : true

    Input : head = [1,2], pos = 0

    Output : true

    Input : head = [1], pos = -1

    Output : false

Leetcode 24 Swap Nodes In Pairs(两两交换链表中的节点)

Posted on 2019-12-16 Edited on 2020-03-05 In leetcode

Difficulty : Medium

Type: Linked List

Language : Java

  1. Clarification

    Given a linked list, swap every two adjacent nodes and return its head
    You may not modify the values in the list’s nodes, only node itself can be changed.

  2. Possible Solutions

    • Iteration

      Set a node,which value is zero, called pre and pre.next points to head.
      Set another node temp equals to pre.

      Set two nodes start and end. temp.next points to start, temp.next.next points to end. In this way, there are only one node or zero node left if start and end are null.
      Basically, we choose three nodes as a group and swap the last two and then we set the new temp to start.
      So, temp.next(pre) -> end(2). start.next(1) -> end.next(3). end.next(2) -> start(1). And now the linked list is pre -> 2 -> 1 -> 3 -> 4 -> null. Set temp = start(1) and begin new swap.

  • Recursion
    This is much tricky. We have to find three important factors to analyze recursion ^_^

    1. When to terminate?
      head is null or head.next is null. It will terminate when there is only one node or no node.
    2. What’s the repetitive action?
      head.next -> next.next
      next.next -> head
      This is the process of swaping.
    3. Return value
      return value is next pointer because next comes before head after swaping.

  1. Code

  1. Test Cases

Given 1->2->3->4, return the list as 2->1->4->3.

Leetcode 206 Reverse Linked List(反转链表)

Posted on 2019-12-16 Edited on 2020-03-05 In leetcode

Difficulty : Easy

Type: Linked List

Language : Java

  1. Clarification

    Reverse a singly linked list

  2. Possible Solutions

    • Iteration
      Set three LinkNode prev, curr and temp. prev and temp are null at first and curr equals to head. prev is the pointer that point to last node. Each time we make curr.next point to previous node and set the temp to curr.next, prev to curr, curr to temp. So that each time the node in linked list will point to previous node and reverse the whole list in the end.
    • Reversion
      head.next.next=head means that head.next point to null at the first and add a next pointer to point itself. So, head.next.next point to head. (So hard to explain it by words - -. There is an animation to show the process)

  3. Code

  1. Test Cases

    输入: 1->2->3->4->5->NULL
    输出: 5->4->3->2->1->NULL

    No duplicate triplets

  2. Possible Solutions

    • Two Pointers
      This question actually use three pointers. The result is to find all pairs of value a and b to make sure a+b=-c. Another pointer is used to record the c.

Leetcode 70 Climbing Stairs(爬楼梯)

Posted on 2019-12-16 Edited on 2020-03-05 In leetcode

Difficulty : Easy

Type: Dynamic programming

Language : Java

  1. Clarification
    Each time you can either climb 1 step or 2 steps. Find all the distinct ways to climb to the top

  2. Possible Solutions

    • 斐波那契数列
      当n <= 2时，结果将为n。 当n> 2时（例如n = 4），到达顶部的最后一步要么是1步要么是2步。 因此，n = one_step_before + two_steps_before。 这是要进行递归的三个指针。 使用for循环更新它们的值并返回最后的n。

    • 动态规划
      分成多个子问题，爬第n阶楼梯的方法数量，等于2部分之和

      • 爬上n - 1阶楼梯的方法数量，因为再爬1阶就能到第n阶
      • 爬上n - 2阶楼梯的方法数量，因为再爬2阶就能到第n阶

      所以公式为dp[n] = dp[n-1] + dp[n-2]

      同时需要初始化dp[0] = 1和dp[1] = 1

  3. Code

    Time Complexity: O(N)

  1. Test Case

    Input : 2

    Output : 2

    Input : 7

    Output : 21

Leetcode 11 Container With Most Water(盛最多水的容器)

Posted on 2019-12-16 Edited on 2020-03-05 In leetcode

Difficulty : Medium

Type: Array && Two pointers

Language : Java

  1. Clarification

    Calculate the max area, which length is the difference of index in the array and the height is the difference of array[index].

  2. Possible Solutions

    • Brute Force

      Use two nested for loop to iterating through array. Calculate the max area then.

    • Two Pointers

      1. Set a left pointer that points to the head of the array and a right pointer that points to the tail of the array. This is to ensure the length changes from maximum to minimum.
      2. If next height is lower than the value of left/right pointer, area will only be smaller. If height[left] < height[right], left pointer increase one. Otherwise, right pointer increase one. This is to find the minimum height. Calculate the area based on current height and move left pointer and right pointer to the middle in the end.

  3. Code

    Time Complexity: O(N^2)

    Time Complexity: O(N)

  4. Test Cases

    Input: [1,8,6,2,5,4,8,3,7]
    Output: 49

Leetcode 283 Move Zeroes（移动0）

Posted on 2019-12-16 Edited on 2020-03-05 In leetcode

Difficulty : Easy

Type: Array && Two pointers

Language : Java

  1. Clarification

    Move all 0’s to the end of the array

    You must do this in-place without making a copy of the array
    Minimize the total number of operations

  2. Possible Solutions

    • Use index

      1. Set an index to zero

      2. For each element in array, if element is not 0, array[index++] equals to the current element. This is to move non-zero element to the head of the array.

      3. While array[index++] < array.length, array[index++] equals to 0. Elements in the left side of index are non-zero, so, set all the elements in the right side of index to zero.

  • Snowball

    Go through the array and gather all zeros on our road

    The initial size of snowball is 1 when we meet first zero in the array.

    If next element of array is not zero, swap it with snowball

    If we encounter another zero, the size of snowball increase 1

  1. Code

Time complexity: O(N)

Time complexity: O(N)

  1. Test cases

    Input: [0,1,0,3,12]
    Output: [1,3,12,0,0]

Leetcode 125 Valid Palindrome（验证回文串)

Posted on 2019-12-14 Edited on 2020-03-06 In leetcode

难度：Easy

类型: String && Two pointers

解题语言： Java

题目解析：
1.Clarification

题目要求验证字符串是否为回文串，只考虑字母和数字字符，可以忽略字母的大小写

2.Possible solutions
- 双指针法  

根据题意，需要先将字符串中的空格，符号等消掉，只比较字母和数字字符，可通过isLetterOrDigit方法进行判断，也可以通过写正则表达式进行判断（运行时间较长）。  

忽略字母大小写可直接用toLowerCase方法将所有字母转为小写。  

设置左指针和右指针，左指针初始值为0，右指针初始值为字符串长度-1，如果左指针小于右指针，且当前指针指向的字符不是字母和数字字符，左指针指向下一个字符（右指针同理，只不过右指针指向前一个字符）。  

如果左指针指向的字符不等于右指针指向的字符，说明不是回文串，返回false，否则左指针指向下一个字符，右指针指向前一个，循环结束后返回true。  

3.Code


时间复杂度O(N)

4.Test Case

测试用例：  
输入: “A man, a plan, a canal: Panama”
输出: true  
输入: “race a car”
输出: false
