Leetcode 103 Binary Tree Zigzag Level Order Traversal(二叉树的锯齿形层次遍历)

Posted on 2020-02-10 Edited on 2020-03-08 In leetcode

Difficulty : Medium

Type: Tree, BFS

Language : Java

  1. Clarification

    Given a binary tree, return the zigzag level order traversal of its nodes’ values. (ie, from left to right, then right to left for the next level and alternate between).

  2. Possible Solutions

    • 递归
      思路同102，设计递归方程helper
      1. root 为空，返回空
      2. res.size() 等于当前深度，res.add一个链表（用于存储当前层的结点值）
      3. 如果链表深度是偶数(depth % 2 ==0), res.get(depth).add(root.val) 获取当前深度的链表，添加当前根结点的值，如果链表深度是奇数，res.get(depth).add(0，root.val)，插入链表时每次都插在0这个index，即每次都插到链表头部，实现从右往左层次遍历加值
      4. 深度depth+1，把root.left作为参数传入下一个递归方程（root.right同理）

  3. Code

Type: Math

Language : Java

    Determine whether an integer is a palindrome. An integer is a palindrome when it reads the same backward as forward.

    Follow up:

    Coud you solve it without converting the integer to a string?

  2. Possible Solutions

    • 每次循环去掉首尾
      首先用一个数字记录数字的最高位，比如312213，可以标记div为100000，首位即为312213/100000 = 3，末位可用余数得到，312213 % 10 = 3，接下来要看1221是否为回文，去掉首位的方法是让312213 % 100000 = 12213，去掉末位的方法是让12213 / 10 = 1221，最后把div也去掉两位，div / 100 = 1000，如此循环，若每次比较的两位都相等，返回true。

Leetcode 144 Binary Tree Preorder Traversal（二叉树的前序遍历）

Posted on 2020-02-01 Edited on 2020-03-09 In leetcode

Difficulty : Medium

Type: Tree, Stack

Language : Java

Clarification

Given a binary tree, return the preorder traversal of its nodes’ values.

Possible Solutions

  • 递归
    需要构建一个辅助方法helper，该方法返回类型为空，递归终止条件是当前的root为空，只要root不为空，就将其值加到list里，通过递归遍历root的左右结点。最后返回添加所有结点值后的list

  • 迭代
    构建一个双端队列deque，新建一个虚拟结点p = root，只要p不为空或者deque不为空，只要p不为空，就往list里加当前p.val，并且把p加到deque里，把p的左结点赋值给p，如果p为空，p就等于deque出队元素的右结点（这一步操作相当于发现当前深度的结点为null，退回到上一个结点，再观察其右结点的情况，直到deque为空且p为空时，循环结束）

Leetcode 104 Maximum Depth of Binary Tree（二叉树的最大深度）

Posted on 2020-01-27 Edited on 2020-03-10 In leetcode

Difficulty : Easy

Type: Tree && Depth-first Search

Language : Java

Clarification

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

Note: A leaf is a node with no children.

Possible Solutions

  • 递归
    DFS思想一次性找到左右子树的深度，比较左右子树深度大小，返回，因为每个结点只访问一次，时间复杂度O(N)，N是结点的数量。空间复杂度在最坏情况下是O(N)，即每个结点只剩下左结点，最好情况下（树完全平衡），空间复杂度O(log(N))
  • 迭代
    BFS思想，记录二叉树的层数，遍历完，层数为最大深度。时间复杂度O(N)，使用队列实现。

Code

Test Cases

example:

Given binary tree [3,9,20,null,null,15,7],

return its depth = 3.
