Leetcode 94 Binary Tree Inorder Traversal(二叉树的中序遍历)

Posted on 2020-02-10 Edited on 2020-03-05 In leetcode

Difficulty : Medium

Type: Tree, Stack

Language : Java

  1. Clarification

    Given a binary tree, return the inorder traversal of its nodes’ values.

  2. Possible Solutions

    • 递归
      需要构建一个辅助方法helper，该方法返回类型为空，递归条件是root为空时直接返回，每次先递归左子树，数据进行加值操作，再递归右子树，即保证中序遍历。

    • 非递归

      通过栈的方式来实现，用一个TreeNode指针模拟访问过程，只要指针不为空，一直依次入栈左子树的值，直到指针为空，再弹出栈顶值，加到数组，再把指针置于弹出值的右子树结点。

Leetcode 145 Binary Tree Postorder Traversal(二叉树的后序遍历)

Posted on 2020-02-10 Edited on 2020-03-05 In leetcode

Difficulty : Hard

Type: Tree, Stack

Language : Java

  1. Clarification

    Given a binary tree, return the postorder traversal of its nodes’ values.

  2. Possible Solutions

    • 递归
      需要构建一个辅助方法helper，该方法返回类型为空，递归条件是root为空时直接返回，每次先递归左子树再递归右子树，数组最后进行加值操作，保证了从后往前遍历。

    • 非递归

      通过栈的方式来实现，首先后序遍历顺序是left -> right -> root，构建一个栈来存储结点，第一次先把根结点入栈，只要栈不为空，就把栈顶结点出栈，如果当前栈顶结点是空就跳过后续步骤直接进入下一次循环，如果当前栈顶结点不为空，将结点的value加到数组里，然后先入栈左子树，再入栈右子树，保证出栈时是先出栈右边的再出栈左边的，最后用Collections.reverse()方法翻转数组，得到结果

Leetcode 103 Binary Tree Zigzag Level Order Traversal(二叉树的锯齿形层次遍历)

Posted on 2020-02-10 Edited on 2020-03-05 In leetcode

Difficulty : Medium

Type: Tree, BFS

Language : Java

  1. Clarification

    Given a binary tree, return the zigzag level order traversal of its nodes’ values. (ie, from left to right, then right to left for the next level and alternate between).

  2. Possible Solutions

    • 递归
      思路同102，设计递归方程helper
      1. root 为空，返回空
      2. res.size() 等于当前深度，res.add一个链表（用于存储当前层的结点值）
      3. 如果链表深度是偶数(depth % 2 ==0), res.get(depth).add(root.val) 获取当前深度的链表，添加当前根结点的值，如果链表深度是奇数，res.get(depth).add(0，root.val)，插入链表时每次都插在0这个index，即每次都插到链表头部，实现从右往左层次遍历加值
      4. 深度depth+1，把root.left作为参数传入下一个递归方程（root.right同理）

Leetcode 144 Binary Tree Preorder Traversal（二叉树的前序遍历）

Posted on 2020-02-01 Edited on 2020-03-05 In leetcode

Difficulty : Medium

Type: Tree, Stack

Language : Java

  1. Clarification

    Given a binary tree, return the preorder traversal of its nodes’ values.

  2. Possible Solutions

    • 递归
      需要构建一个辅助方法helper，该方法返回类型为空，递归终止条件是当前的root为空，只要root不为空，就将其值加到list里，通过递归遍历root的左右结点。最后返回添加所有结点值后的list

    • 迭代
      构建一个双端队列deque，新建一个虚拟结点p = root，只要p不为空或者deque不为空，只要p不为空，就往list里加当前p.val，并且把p加到deque里，把p的左结点赋值给p，如果p为空，p就等于deque出队元素的右结点（这一步操作相当于发现当前深度的结点为null，退回到上一个结点，再观察其右结点的情况，直到deque为空且p为空时，循环结束）
