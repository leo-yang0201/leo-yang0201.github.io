Leetcode 62 Unique Paths(不同路径)

Posted on 2020-02-15 Edited on 2020-03-05 In leetcode

Difficulty : Medium

Type: Dynamic Programming

Language : Java

  1. Clarification

    A robot is located at the top-left corner of a m x n grid (marked ‘Start’ in the diagram below).

    The robot can only move either down or right at any point in time. The robot is trying to reach the bottom-right corner of the grid (marked ‘Finish’ in the diagram below).

    How many possible unique paths are there?

    Note: m and n will be at most 100.

  2. Possible Solutions

    • 动态规划
      • 机器人每次只能向右或向下移动一步
        • 左上角到右下角的走法 = 从右边开始走的路径总数+从下边开始走的路径总数
        • 动态方程
          • dp[ i ][ j ] = dp[i - 1][ j ] + dp[ i ][ j - 1]
          • 初始化第一行和第一列的值
            • dp[0][ j ] = 1, dp[i][0] = 1
            • 因为一直向下或者一直向右走而不转向只有一种走法

Posted on 2020-02-10 Edited on 2020-03-08 In leetcode

Difficulty : Medium

Type: Tree, Stack

Language : Java

  1. Clarification

    Given a binary tree, return the inorder traversal of its nodes’ values.

  2. Possible Solutions

    • 递归
      需要构建一个辅助方法helper，该方法返回类型为空，递归条件是root为空时直接返回，每次先递归左子树，数据进行加值操作，再递归右子树，即保证中序遍历。

    • 非递归

      通过栈的方式来实现，用一个TreeNode指针模拟访问过程，只要指针不为空，一直依次入栈左子树的值，直到指针为空，再弹出栈顶值，加到数组，再把指针置于弹出值的右子树结点。

Leetcode 145 Binary Tree Postorder Traversal(二叉树的后序遍历)

Posted on 2020-02-10 Edited on 2020-03-08 In leetcode

Difficulty : Hard

Type: Tree, Stack

Language : Java

  1. Clarification

    Given a binary tree, return the postorder traversal of its nodes’ values.

  2. Possible Solutions

    • 递归
      需要构建一个辅助方法helper，该方法返回类型为空，递归条件是root为空时直接返回，每次先递归左子树再递归右子树，数组最后进行加值操作，保证了从后往前遍历。

    • 非递归

      通过栈的方式来实现，首先后序遍历顺序是left -> right -> root，构建一个栈来存储结点，第一次先把根结点入栈，只要栈不为空，就把栈顶结点出栈，如果当前栈顶结点是空就跳过后续步骤直接进入下一次循环，如果当前栈顶结点不为空，将结点的value加到数组里，然后先入栈左子树，再入栈右子树，保证出栈时是先出栈右边的再出栈左边的，最后用Collections.reverse()方法翻转数组，得到结果

Type: Tree, BFS

Language : Java

  1. Clarification

    Given a binary tree, return the zigzag level order traversal of its nodes’ values. (ie, from left to right, then right to left for the next level and alternate between).

  2. Possible Solutions

    • 递归
      思路同102，设计递归方程helper
      1. root 为空，返回空
      2. res.size() 等于当前深度，res.add一个链表（用于存储当前层的结点值）

      4. 深度depth+1，把root.left作为参数传入下一个递归方程（root.right同理）

Type: Math

Language : Java

  1. Clarification

    Determine whether an integer is a palindrome. An integer is a palindrome when it reads the same backward as forward.

    Follow up:

    Coud you solve it without converting the integer to a string?

  2. Possible Solutions

    • 每次循环去掉首尾
      首先用一个数字记录数字的最高位，比如312213，可以标记div为100000，首位即为312213/100000 = 3，末位可用余数得到，312213 % 10 = 3，接下来要看1221是否为回文，去掉首位的方法是让312213 % 100000 = 12213，去掉末位的方法是让12213 / 10 = 1221，最后把div也去掉两位，div / 100 = 1000，如此循环，若每次比较的两位都相等，返回true。

Leetcode 144 Binary Tree Preorder Traversal（二叉树的前序遍历）

Posted on 2020-02-01 Edited on 2020-03-10 In leetcode

Difficulty : Medium

Type: Tree, Stack

Language : Java

Clarification

Given a binary tree, return the preorder traversal of its nodes’ values.

Possible Solutions

  • 递归
    需要构建一个辅助方法helper，该方法返回类型为空，递归终止条件是当前的root为空，只要root不为空，就将其值加到list里，通过递归遍历root的左右结点。最后返回添加所有结点值后的list

  • 迭代
    构建一个双端队列deque，新建一个虚拟结点p = root，只要p不为空或者deque不为空，只要p不为空，就往list里加当前p.val，并且把p加到deque里，把p的左结点赋值给p，如果p为空，p就等于deque出队元素的右结点（这一步操作相当于发现当前深度的结点为null，退回到上一个结点，再观察其右结点的情况，直到deque为空且p为空时，循环结束）
