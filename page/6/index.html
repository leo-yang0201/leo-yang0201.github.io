Leetcode 1103 distribute candies to people(分糖果II)

Difficulty : Medium

Type: LinkedList

Language : Java

Clarification

We distribute some number of candies, to a row of n = num_people people in the following way:

We then give 1 candy to the first person, 2 candies to the second person, and so on until we give n candies to the last person.

Then, we go back to the start of the row, giving n + 1 candies to the first person, n + 2 candies to the second person, and so on until we give 2 * n candies to the last person.

This process repeats (with us giving one more candy each time, and moving to the start of the row after we reach the end) until we run out of candies. The last person will receive all of our remaining candies (not necessarily one more than the previous gift).

Return an array (of length num_people and sum candies) that represents the final distribution of candies.

Constraints:

  • 1 <= candies <= 10^9
  • 1 <= num_people <= 1000

Possible Solutions

  • 暴力法
    不断地遍历数组，如果还有糖就一直分，直到没有糖为止。

  • 等差数列

    除了最后一份礼物数量是根据还剩多少糖果决定，剩下礼物数量都是从1开始构成的等差数列，所以思路是根据等差数列求和公式找到每次还剩多少candy，只要不是最后一行，每次要加的值都可以根据求和公式算出来然后叠加到数组位置上，到了最后一行最后一个位置的时候直接把剩余糖果数目累加到该位置

Leetcode 82 remove-duplicates-from-sorted-list-ii(删除排序链表中的重复元素 II)

Difficulty : Medium

Type: LinkedList

Language : Java

  1. Clarification

    Given a sorted linked list, delete all nodes that have duplicate numbers, leaving only distinct numbers from the original list.

  2. Possible Solutions

    这道题仍然是Expedia的一道面经题目

思路：借鉴（windliang大佬的讲解）

  • 迭代
    只需要两个指针，一个指针 pre 代表重复数字的前边的一个指针，另一个指针 cur 用来遍历链表。d 代表哨兵节点，用来简化边界条件，初始化为 head 指针的前一个节点。p 代表 pre，c 代表 cur。

    时间复杂度O(N)，空间复杂度O(1)

  • 递归

    分头结点和后边的结点相等的情况和不等的情况

    这道题自己卡住是因为开始的时候考虑用栈来解决，每次都入栈，然后如果栈顶元素等于下一个要入栈的元素，就不入栈并且pop掉栈顶元素，但是这么做似乎没办法处理重复元素个数超过两个的情况，其实还是考察链表指针的操作，自己想的方向不对

Leetcode 1328 Break a Palindrome(破坏回文串)

Difficulty : Medium

Type: String

Language : Java

  1. Clarification

    Given a palindromic string palindrome, replace exactly one character by any lowercase English letter so that the string becomes the lexicographically smallest possible string that isn’t a palindrome.

    After doing so, return the final string. If there is no way to do so, return the empty string.

  2. Possible Solutions

    这道题是Expedia的一道面经题目，最近投了他家的SDE实习，准备一下

思路：

  • 字符串长度为1的时候返回空串
  • 否则检查前面一半长度的字符，如果出现第一个不是a的字符就把它改成a，返回修改后的字符串
  • 如果全都是a，如”aaaa”，把最后一个字符改成b，返回修改后的字符串

这道题自己卡住是因为一直在想如何操作字典序使其最小，想复杂了，就换成a就行了，需要考虑到全是a和只有一个字符的情况。

Leetcode 64 Minimum Path Sum(最小路径和)

Difficulty : Medium

Type: Dynamic Programming

Language : Java

  1. ClarificationGiven a m x n grid filled with non-negative numbers, find a path from top left to bottom right which minimizes the sum of all numbers along its path.

    Note: You can only move either down or right at any point in time.

  1. Possible Solutions

    • 动态规划
      题目要求，只能向右或者向下走，意味着当前单元格(i,j)只能从左单元格(i-1,j)和上单元格(i,j-1)走到，因此只需要考虑矩阵左边界和上边界
      • 走到当前单元格(i,j)的最小路径和 = 从左方单元格(i-1,j)与从上方单元格(i,j-1)走来的两个最小路径和中较小的+当前单元格值，具体有4种情况
        • 左边和上边都不是矩阵边界(i!=0,j!=0),dp[i][j] = min(dp[i-1][j],dp[i][j-1]) + dp[i][j]
        • 只有左边是边界时，只能从上边来,dp[i][j] = dp[i][j-1] + dp[i][j
        • 只有上边是边界时，只能从左边来,dp[i][j] = dp[i-1][j] + dp[i][j]
        • 左边和上边都是边界，就是起点，dp[i][j] = grid[i][j]

Leetcode 94 Binary Tree Inorder Traversal(二叉树的中序遍历)

Difficulty : Medium

Type: Tree, Stack

Language : Java

  1. Clarification

    Given a binary tree, return the inorder traversal of its nodes’ values.

  2. Possible Solutions

    • 递归
      需要构建一个辅助方法helper，该方法返回类型为空，递归条件是root为空时直接返回，每次先递归左子树，数据进行加值操作，再递归右子树，即保证中序遍历。

    • 非递归

      通过栈的方式来实现，用一个TreeNode指针模拟访问过程，只要指针不为空，一直依次入栈左子树的值，直到指针为空，再弹出栈顶值，加到数组，再把指针置于弹出值的右子树结点。

Leetcode 145 Binary Tree Postorder Traversal(二叉树的后序遍历)

Difficulty : Hard

Type: Tree, Stack

Language : Java

  1. Clarification

    Given a binary tree, return the postorder traversal of its nodes’ values.

  2. Possible Solutions

    • 递归
      需要构建一个辅助方法helper，该方法返回类型为空，递归条件是root为空时直接返回，每次先递归左子树再递归右子树，数组最后进行加值操作，保证了从后往前遍历。

    • 非递归

      通过栈的方式来实现，首先后序遍历顺序是left -> right -> root，构建一个栈来存储结点，第一次先把根结点入栈，只要栈不为空，就把栈顶结点出栈，如果当前栈顶结点是空就跳过后续步骤直接进入下一次循环，如果当前栈顶结点不为空，将结点的value加到数组里，然后先入栈左子树，再入栈右子树，保证出栈时是先出栈右边的再出栈左边的，最后用Collections.reverse()方法翻转数组，得到结果
