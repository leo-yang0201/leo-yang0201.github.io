Leetcode 9 Palindrome Number(回文数)

Posted on 2020-02-03 Edited on 2020-03-05 In leetcode

Type: Math

Language : Java

  1. Clarification

    Determine whether an integer is a palindrome. An integer is a palindrome when it reads the same backward as forward.

    Follow up:

    Coud you solve it without converting the integer to a string?

  2. Possible Solutions

    • 每次循环去掉首尾
      首先用一个数字记录数字的最高位，比如312213，可以标记div为100000，首位即为312213/100000 = 3，末位可用余数得到，312213 % 10 = 3，接下来要看1221是否为回文，去掉首位的方法是让312213 % 100000 = 12213，去掉末位的方法是让12213 / 10 = 1221，最后把div也去掉两位，div / 100 = 1000，如此循环，若每次比较的两位都相等，返回true。

Leetcode 144 Binary Tree Preorder Traversal（二叉树的前序遍历）

Posted on 2020-02-01 Edited on 2020-03-10 In leetcode

Difficulty : Medium

Type: Tree, Stack

Language : Java

Clarification

Given a binary tree, return the preorder traversal of its nodes’ values.

Possible Solutions

  • 递归
    需要构建一个辅助方法helper，该方法返回类型为空，递归终止条件是当前的root为空，只要root不为空，就将其值加到list里，通过递归遍历root的左右结点。最后返回添加所有结点值后的list

  • 迭代
    构建一个双端队列deque，新建一个虚拟结点p = root，只要p不为空或者deque不为空，只要p不为空，就往list里加当前p.val，并且把p加到deque里，把p的左结点赋值给p，如果p为空，p就等于deque出队元素的右结点（这一步操作相当于发现当前深度的结点为null，退回到上一个结点，再观察其右结点的情况，直到deque为空且p为空时，循环结束）

Leetcode 104 Maximum Depth of Binary Tree（二叉树的最大深度）

Posted on 2020-01-27 Edited on 2020-03-10 In leetcode

Difficulty : Easy

Type: Tree && Depth-first Search

Language : Java

Clarification

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

Note: A leaf is a node with no children.

Possible Solutions

  • 递归
    DFS思想一次性找到左右子树的深度，比较左右子树深度大小，返回，因为每个结点只访问一次，时间复杂度O(N)，N是结点的数量。空间复杂度在最坏情况下是O(N)，即每个结点只剩下左结点，最好情况下（树完全平衡），空间复杂度O(log(N))
  • 迭代
    BFS思想，记录二叉树的层数，遍历完，层数为最大深度。时间复杂度O(N)，使用队列实现。

Code

Test Cases

example:

Given binary tree [3,9,20,null,null,15,7],

return its depth = 3.

Leetcode 225 Implement Stack with Queue(用队列实现栈)

Posted on 2020-01-22 Edited on 2020-03-05 In leetcode

Difficulty : Easy

Type: Stack,Design

Language : Java

  1. Clarification

    • Implement the following operations of a stack using queues.
      • push(x) – Push element x onto stack.
      • pop() – Removes the element on top of the stack.
      • top() – Get the top element.
      • empty() – Return whether the stack is empty.

    Notes:

    • You must use only standard operations of a queue – which means only push to back, peek/pop from front, size, and is empty operations are valid.
    • Depending on your language, queue may not be supported natively. You may simulate a queue by using a list or deque (double-ended queue), as long as you use only standard operations of a queue.
    • You may assume that all operations are valid (for example, no pop or top operations will be called on an empty stack).

  2. Possible Solutions

    • 出队入队操作
      入队时，先把当前值入队，再把顺序倒成栈的样子，具体操作是将当前出队的值重新入队，这个值就会从队列头移到队列尾，每次移动队列长度 - 1 次。

  3. Code

  4. Test Cases

    MyStack stack = new MyStack();

    stack.push(1);
    stack.push(2);
    stack.top(); // returns 2
    stack.pop(); // returns 2
    stack.empty(); // returns false

Leetcode 232 Implement Queue using Stacks(用栈实现队列)

Posted on 2020-01-21 Edited on 2020-03-05 In leetcode

Difficulty : Easy

Type: Stack,Queue

Language : Java

  1. Clarification

    Implement the following operations of a queue using stacks.

    • push(x) – Push element x to the back of queue.
    • pop() – Removes the element from in front of queue.
    • peek() – Get the front element.
    • empty() – Return whether the queue is empty.

    Notes:

    • You must use only standard operations of a stack – which means only push to top, peek/pop from top, size, and is empty operations are valid.
    • Depending on your language, stack may not be supported natively. You may simulate a stack by using a list or deque (double-ended queue), as long as you use only standard operations of a stack.
    • You may assume that all operations are valid (for example, no pop or peek operations will be called on an empty queue).

  2. Possible Solutions

    • 双栈解法
      一个输入栈，一个输出栈，数据push时进输入栈，输出的时候只要把输入栈的数据入栈输出栈，再pop输出栈就倒过来成队列了

  3. Code

  4. Test Cases

    MyQueue queue = new MyQueue();

    queue.push(1);
    queue.push(2);
    queue.peek(); // 返回 1
    queue.pop(); // 返回 1
    queue.empty(); // 返回 false
