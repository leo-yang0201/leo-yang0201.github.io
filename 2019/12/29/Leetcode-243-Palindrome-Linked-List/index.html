Leetcode 243 Palindrome Linked List(回文链表)

发表于 2019-12-29

Difficulty : Easy

Type: Linked List, Two Pointers

Language : Java

  1. Clarification

    Given a singly linked list, determine if it is a palindrome. 要求使用O(n)时间复杂度和O(1)空间复杂度

  2. Possible Solutions

    • 快慢指针
      1. 回文是轴对称的，所以思路是先找到轴，然后把轴右侧的链表进行反转，再遍历比较两个链表的值是不是一致，都是一致的就是回文
      2. 快指针每次走两个，慢指针每次走一个，这样快指针为空或者快指针的下一个结点为空时，慢指针正好走到轴心
      3. 此时慢指针在中心点，破坏链表，让慢指针的下一个结点指向null，再把慢指针放回到第一个结点处，为后续遍历比较作准备
      4. 把后半段的链表进行反转，方法是设置一个空结点pre，先用temp结点储存当前结点cur的下一个结点，再让cur指向pre，再让pre=cur，cur=temp，最后pre结点就是反转后链表的头结点
      5. 比较slow结点和pre结点的值是否相等，相等就继续迭代（注：结点数为奇数时，左边会比右边多一个结点，所以从右边开始迭代），都相等就返回true，否则false

  3. Code

  1. Test Cases

    Input: 1->2->2->1
    Output: true
