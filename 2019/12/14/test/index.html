Leetcode 125 验证回文串

2019-12-14 • 标签: leetcode

难度：Easy

类型: Array

解题语言： Java

题目解析：
1.Clarification
题目要求验证字符串是否为回文串，只考虑字母和数字字符，可以忽略字母的大小写
2.Possible solutions
- 双指针法
根据题意，需要先将字符串中的空格，符号等消掉，只比较字母和数字字符，可通过isLetterOrDigit方法进行判断，也可以通过写正则表达式进行判断（运行时间较长）。忽略字母大小写可直接用toLowerCase方法将所有字母转为小写。

设置左指针和右指针，左指针初始值为0，右指针初始值为字符串长度-1，如果左指针小于右指针，且当前指针指向的字符不是字母和数字字符，左指针指向下一个字符（右指针同理，只不过右指针指向前一个字符）。如果左指针指向的字符不等于右指针指向的字符，返回false，否则左指针指向下一个字符，右指针指向前一个，循环结束后返回true

3.Code

class Solution {
public boolean isPalindrome(String s) {
int left=0;int right=s.length()-1;
while(left<right){
while(left<right && !Character.isLetterOrDigit(s.charAt(left))){left++;}
while(left<right && !Character.isLetterOrDigit(s.charAt(right))){right–;}
if(Character.toLowerCase(s.charAt(left)) != Character.toLowerCase(s.charAt(right))) return false;
left++;right–;
}
return true;
}
}
