Leetcode 189 Rotate Array (旋转数组)

Posted on 2019-12-19 Edited on 2020-03-05 In leetcode Views:

Difficulty : Easy

Type: Array

Language : Java

  1. Clarification

    Come up as many solutions as you can.

    Do it in-place with O(1) extra space

  2. Possible Solutions

    • Brute Force

      1. Save last element as temp
      2. For each element in array, each element equals to previous element.
      3. First element equals to temp
        • Problems
          Time Complexity is O(n*k). So, its runtime will be longer in this way.

    • Reverse

      1. Reverse each element in the array
      2. Elements which need to move to the head are the remainder of k and nums.length
      3. Thus, reverse k elements and then reverse n - k elements.

  3. Code

Time complexity: O(N*K)

Time complexity: O(N)

  1. Test cases

    Input: [1,2,3,4,5,6,7],k=3
    Output: [5,6,7,1,2,3,4]
