Leetcode 21 Merge Two Sorted lists(合并两个有序链表)

Posted on 2019-12-19 Edited on 2020-02-10

Difficulty : Easy

Type: Linked List

Language : Java

  1. Clarification

    Come up as many solutions as you can.

    Do it in-place with O(1) extra space

  2. Possible Solutions

    • Recursion

      1. If one link list is empty, return another one
      2. Compare l1’s value with l2’s value, if l1 is smaller then use recursion on l1.next and l2, return l1 then.
        Else use recursion on l2.next and l1, return l2 then.

    • Iteration

      1. Create a new linked list
      2. Compare l1’s value with l2’s value, link the smaller one with new linked list
      3. If one linked list is empty whereas another is not empty. Link the whole unvoid list to the new linked list

  3. Code

  1. Test cases

    Input: [1,2,4],[1,3,4]
    Output: [1,1,2,3,4,4]
