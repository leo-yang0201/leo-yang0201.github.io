Leetcode 11 Container With Most Water(盛最多水的容器)

2019-12-16 • 分类: leetcode • 标签: Array, Two Pointers

Difficulty : Medium

Type: Array && Two pointers

Language : Java

  1. Clarification

    Calculate the max area, which length is the difference of index in the array and the height is the difference of array[index].

  2. Possible Solutions

    • Brute Force

      Use two nested for loop to iterating through array. Calculate the max area then.

    • Two Pointers

      1. Set a left pointer that points to the head of the array and a right pointer that points to the tail of the array. This is to ensure the length changes from maximum to minimum.
      2. If next height is lower than the value of left/right pointer, area will only be smaller. If height[left] < height[right], left pointer increase one. Otherwise, right pointer increase one. This is to find the minimum height. Calculate the area based on current height and move left pointer and right pointer to the middle in the end.

  3. Code

    Time Complexity: O(N^2)

    Time Complexity: O(N)

  4. Test Cases

    Input: [1,8,6,2,5,4,8,3,7]
    Output: 49
