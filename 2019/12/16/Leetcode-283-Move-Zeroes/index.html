Leetcode 283 Move Zeroes（移动0）

2019-12-16 • 标签: leetcode

Difficulty : Easy

Type: Array && Two pointers

Language : Java

  1. Clarification

    Move all 0’s to the end of the array

    You must do this in-place without making a copy of the array
    Minimize the total number of operations

  2. Possible Solutions

    • Use index
      1. Set an index to zero
      2. For each element in array, if element is not 0, array[index++] equals to the current element. This is to move non-zero element to the head of the array.
      3. While array[index++] < array.length, array[index++] equals to 0. Elements in the left side of index are non-zero, so, set all the elements in the right side of index to zero.

  3. Code

    Time complexity: O(N)

  4. Test cases

    Input: [0,1,0,3,12]
    Output: [1,3,12,0,0]
