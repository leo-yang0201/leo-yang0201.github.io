Leetcode 203 Remove Linked List Elements(移除链表元素)

发表于 2019-12-26

Difficulty : Easy

Type: Linked List

Language : Java

  1. Clarification

    Remove all elements from a linked list of integers that have value val\.

  2. Possible Solutions

    • Iteration(迭代)
      注意头节点情况，如果头节点需要被删除，需要把head结点移至head.next。设置两个结点prev和cur，cur指向head，prev为cur的前驱结点，删除时，prev.next -> cur.next，再令cur = cur.next即可，若当前遍历结点不需删除，prev指向cur，cur指向cur.next
    • Reversion（递归）
      • 将链表分解为头结点和一个更小的链表
      • 在更小的链表中删除给定的元素，将新链表res返回
      • 然后判断头结点应不应该被删除，如果应该被删除，就返回新链表res，如果不该被删除，就把新链表链在head后，返回head
      • 不能再小的基本问题是：链表缩短到不能再短，即链表中没有结点，此时head == null

  3. Code

  1. Test Cases

    Input: 1->2->6->3->4->5->6, val = 6
    Output: 1->2->3->4->5
