Leetcode 350 Intersection of Two Arrays II(两个数组的交集II)

Posted on 2020-03-26 In leetcode

Difficulty : Easy

Type: Hash Table,Two Pointers,Binary Search,Sort

Language : Java

Clarification

Given two arrays, write a function to compute their intersection.

Note:

  • Each element in the result should appear as many times as it shows in both arrays.
  • The result can be in any order.

Follow up:

  • What if the given array is already sorted? How would you optimize your algorithm?
  • What if nums1‘s size is small compared to nums2‘s size? Which algorithm is better?
  • What if elements of nums2 are stored on disk, and the memory is limited such that you cannot load all elements into the memory at once?

Possible Solutions

  • 哈希表
    HashMap进行映射，先用HashMap记录第一个数组中的元素到key里，每个元素出现的次数放在value里。然后遍历第二个数组，若找到对应元素且该元素出现的次数大于0，就添加这个元素到list里，并且HashMap的value - 1，表示已经找到一个相同的了。最后把list的值遍历取出赋值给数组。
  • 双指针
    先对两个数组进行排序，然后设双指针分别指向第一个数组和第二个数组的头部，如果nums[p1] == nums[p2]，说明两个数组都有这个数，加入list中。如果不等，移动小的那个指针。

Leetcode 200 Number of Islands(岛屿数量)

Posted on 2020-03-25 In leetcode

Difficulty : Medium

Type: DFS,BFS,Union Find

Language : Java

Clarification

Given a 2d grid map of '1's (land) and '0's (water), count the number of islands. An island is surrounded by water and is formed by connecting adjacent lands horizontally or vertically. You may assume all four edges of the grid are all surrounded by water.

Possible Solutions

  • 深度优先遍历

    两层循环遍历二维数组里所有的点，只要是陆地(二维数组当前坐标是1)，就进行DFS深度遍历，将当前点坐标转为0(视为已经标记过)，通过递归查看当前点的前后左右四个方向还有没有陆地，如果都没有，count加1(代表找到一个岛屿)，如果四个方向中有陆地，递归到那块陆地再进行上下左右的判断，直到所有点都标记过。

Leetcode 136 Single Number(只出现一次的数字)

Posted on 2020-03-25 In leetcode

Difficulty : Easy

Type: Hash Table,Bit Manipulation

Language : Java

Clarification

Given a non-empty array of integers, every element appears twice except for one. Find that single one.

Note:

Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Possible Solutions

  • 哈希表

    新建一个Hashmap，两次循环，第一次循环把nums数组里的值作为key，出现次数count作为value依次put进去，第二次循环map的key列表，看哪个key的count是1，返回那个key。

  • 位运算（异或）

    题目要求不用额外空间实现，所以哈希表不是最佳解法，最好的方法是通过位运算里的异或运算。

    异或运算特点：

    • 一个数和0做异或运算等于本身
    • 一个数和其本身做异或运算等于0
    • 异或运算满足交换律和结合律

    将所有数字按顺序做异或运算，由于交换律和结合律，只要出现过两次的数字最后都会变成自己和自己做异或运算，最后就是0和那个只出现一次的数字做异或运算，所以循环后得到的最后计算结果就是只出现一次的数字。

Posted on 2020-03-24 In leetcode

Difficulty : Easy

Type: Dynamic Programming

Language : Java

Clarification

一个有名的按摩师会收到源源不断的预约请求，每个预约都可以选择接或不接。在每次预约服务之间要有休息时间，因此她不能接受相邻的预约。给定一个预约请求序列，替按摩师找到最优的预约集合（总预约时间最长），返回总的分钟数。

Possible Solutions

  • 动态规划
    递推方程: dp[i] = max(dp[i - 1],dp[i - 2] + nums[i])，这道题很像打家劫舍的思路，但是自己还是一下写不出来，动态规划的题目还需要大量反复练习，现在也只有股票的题目算是没问题了。
