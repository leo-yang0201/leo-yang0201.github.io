Leetcode 85 Maximal Rectangle(最大矩形)

2020-01-07 • 分类: leetcode • 标签: Array, Dynamic Programming, Hash Table, Stack

Difficulty : Hard

Type: Stack && Array && Hash Table && Dynamic Programming

Language : Java

  1. Clarification

    Given a 2D binary matrix filled with 0’s and 1’s, find the largest rectangle containing only 1’s and return its area.

  2. Possible Solutions

    • 栈(方法同84题)

      思路和84题完全一样，本题的二维矩阵每一层向上都可以看作一个直方图，输入矩阵有多少行，就可以形成多少个直方图，对每个直方图都掉用84题的算法，再每次都更新最大面积即可。
      遍历每个柱子，分以下情况

      • 如果当前栈空，或者当前柱子大于栈顶柱子的高度，就将当前柱子的下标入栈

      • 当前柱子的高度小于栈顶柱子的高度。那么就把栈顶柱子出栈，而右边第一个小于当前柱子的下标就是当前再遍历的柱子，左边第一个小于当前柱子的下标就是当前新的栈顶

        遍历结束后，如果栈没有空，就依次出栈，出栈元素当作要求面积的柱子，然后依次计算矩形区域面积
