Leetcode 84 Largest Rectangle In Histogram(柱状图中最大的矩形)

Difficulty : Hard

Type: Stack && Array

Language : Java

  1. Clarification

    Given n non-negative integers representing the histogram’s bar height where the width of each bar is 1, find the area of largest rectangle in the histogram.

    Above is a histogram where width of each bar is 1, given height = [2,1,5,6,2,3].

  2. Possible Solutions

    • 暴力法
      以题目的例子，柱形图高度有1，2，3，5，6，只需要找出每一个高度对应的最大面积，选出最大的即可。如果求高度为3面积最大的，需要遍历每一个高度，然后看连续的大于等于3的柱形有几个，如果是n个，那么此时面积就是3*n。所以高度确定之后，只需要找连续的大于等于3的柱形个数，也就是高度。

    • 栈
      遍历每个柱子，分以下情况

      • 如果当前栈空，或者当前柱子大于栈顶柱子的高度，就将当前柱子的下标入栈

      • 当前柱子的高度小于栈顶柱子的高度。那么就把栈顶柱子出栈，而右边第一个小于当前柱子的下标就是当前再遍历的柱子，左边第一个小于当前柱子的下标就是当前新的栈顶

        遍历结束后，如果栈没有空，就依次出栈，出栈元素当作要求面积的柱子，然后依次计算矩形区域面积

  3. Code

  1. Test Cases

    Input : [2,1,5,6,2,3]

    Output : 10
