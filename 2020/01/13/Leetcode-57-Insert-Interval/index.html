Leetcode 57 Insert Interval（插入区间）

2020-01-13 • 标签: Array, Sort, leetcode

Difficulty : Hard

Type: Array && Sort

Language : Java

  1. Clarification

    Given a set of non-overlapping intervals, insert a new interval into the intervals (merge if necessary).

    You may assume that the intervals were initially sorted according to their start times.

  2. Possible Solutions

    • 贪心算法
      这道题思路类似于56题，本题可以运用贪心算法，从最简单的字结构开始拆分。
      • 当interval整体在newInterval整体之前时，直接添加interval整体到arraylist中，并使插入位置 + 1
      • 当interval整体在newInterval整体之后时，说明newInterval整体已经添加成功，直接添加interval整体到arraylist中
      • interval和newInterval存在相交的情况，只要相交，将interval于newInterval融合，左下标取interval和newInterval左下标最小值，右坐标取interval和newInterval右下标最大值，实现两个数组的融合
      • 最后遍历完interval，将newInterval插入到res指定的位置i

  4. Test Cases

    Input: intervals = [[1,2],[3,5],[6,7],[8,10],[12,16]], newInterval = [4,8]
    Output: [[1,2],[3,10],[12,16]]
    Explanation: Because the new interval [4,8] overlaps with [3,5],[6,7],[8,10].
