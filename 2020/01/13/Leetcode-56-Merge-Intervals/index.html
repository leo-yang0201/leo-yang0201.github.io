Leetcode 56 Merge Intervals(合并区间)

Posted on 2020-01-13 Edited on 2020-03-05 In leetcode Views:

Difficulty : Medium

Type: Array && Sort

Language : Java

  1. Clarification

    Given a collection of intervals, merge all overlapping intervals.

  2. Possible Solutions

    • List 转 二维数组
      这道题思路就是要获取区间的左端点和右端点，当右端点大于等于下一个区间的左端点时，只需要比较右端点和下一个区间的右端点谁更大，并将右端点的值更新为两者中的最大值，即完成了合并操作。如果右端点比下一个区间的左端点小，说明两区间没有交集，不需要合并。

  3. Code

  4. Test Cases

    Input: [[1,3],[2,6],[8,10],[15,18]]
    Output: [[1,6],[8,10],[15,18]]
    Explanation: Since intervals [1,3] and [2,6] overlaps, merge them into [1,6].
