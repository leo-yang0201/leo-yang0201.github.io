Leetcode 83 Remove Duplicates from Sorted List(删除排序链表中的重复元素)

2020-01-20 • 标签: LinkedList, leetcode

Difficulty : Easy

Type: Linked List

Language : Java

  1. Clarification

    Given a sorted linked list, delete all duplicates such that each element appear only once.

  2. Possible Solutions

    • 直接判断链表值然后删除结点
      单设一个结点cur，如果cur.next不为空，就继续循环，判断cur.next.val 是不是等于 cur.val，如果相等说明出现重复值，cur.next = cur.next.next，跳过重复值指针，即可。
