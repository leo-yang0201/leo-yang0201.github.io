Leetcode 101 Symmetric Tree（对称二叉树）

2020-01-31 • 分类: leetcode • 标签: BFS, DFS, tree

Difficulty : Easy

Type: Binary Tree, DFS, BFS

Language : Java

  1. Clarification

    Given a binary tree, check whether it is a mirror of itself (ie, symmetric around its center).

  2. Possible Solutions

    • 递归
      注意如果根结点为null，返回true
      如果根结点的左右结点存在且相等，返回true
      如果根结点的左右结点不存在，返回true
      如果根结点左右结点只存在一个，返回false
      如果左右结点都存在但是值不等，返回false
      根据以上条件设计递归方程的终止条件
