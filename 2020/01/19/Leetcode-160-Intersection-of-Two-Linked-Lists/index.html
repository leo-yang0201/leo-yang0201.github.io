Leetcode 160 Intersection of Two Linked Lists(相交链表)

发表于 2020-01-19

Difficulty : Easy

Type: Linked List

Language : Java

  1. Clarification

    Write a program to find the node at which the intersection of two singly linked lists begins.

    Notes:

    • If the two linked lists have no intersection at all, return null.
    • The linked lists must retain their original structure after the function returns.
    • You may assume there are no cycles anywhere in the entire linked structure.
    • Your code should preferably run in O(n) time and use only O(1) memory.

  2. Possible Solutions

    • 双指针
      这道题因为要求用O(1)的时间复杂度，所以不能考虑新建一个Set，存第一个链表的值，再遍历下一个链表找Set中对应值的方法。设n1,n2两个指针，分别放在headA和headB，只遍历一次，如果n1走到null，就把n1放到headB，否则n1继续走，如果n2走到null，就把n2放在headA，否则n2继续走，因为两个链表的长度可能存在差值，先走到null那个指针放回另一个链表的头部后，两个指针只要持续往下走就一定会相交，交点就是这两个链表的交点（思路类似环形链表，只不过是人工进行了环形操作）

    Input: intersectVal = 2, listA = [0,9,1,2,4], listB = [3,2,4], skipA = 3, skipB = 1
    Output: Reference of the node with value = 2
    Input Explanation: The intersected node’s value is 2 (note that this must not be 0 if the two lists intersect). From the head of A, it reads as [0,9,1,2,4]. From the head of B, it reads as [3,2,4]. There are 3 nodes before the intersected node in A; There are 1 node before the intersected node in B.

    Input: intersectVal = 0, listA = [2,6,4], listB = [1,5], skipA = 3, skipB = 2
    Output: null
    Input Explanation: From the head of A, it reads as [2,6,4]. From the head of B, it reads as [1,5]. Since the two lists do not intersect, intersectVal must be 0, while skipA and skipB can be arbitrary values.
    Explanation: The two lists do not intersect, so return null.
