Leetcode 104 Maximum Depth of Binary Tree（二叉树的最大深度）

Posted on 2020-01-27 Edited on 2020-03-10 In leetcode Views:

Difficulty : Easy

Type: Tree && Depth-first Search

Language : Java

Clarification

Given a binary tree, find its maximum depth.

The maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

Note: A leaf is a node with no children.

Possible Solutions

  • 递归
    DFS思想一次性找到左右子树的深度，比较左右子树深度大小，返回，因为每个结点只访问一次，时间复杂度O(N)，N是结点的数量。空间复杂度在最坏情况下是O(N)，即每个结点只剩下左结点，最好情况下（树完全平衡），空间复杂度O(log(N))
  • 迭代
    BFS思想，记录二叉树的层数，遍历完，层数为最大深度。时间复杂度O(N)，使用队列实现。
