Statistics for IT Managers 95-796 notes（知识点汇总）

发表于 2020-01-02

  1. Descriptive Statistics and Probability

    • Basic Definitions 基础概念

      • Statistics is the science of analyzing and interpreting data, transforming raw data into information.
      • Descriptive statistics 描述统计
        • Graphical displays (histograms, box plots) 图表形式呈现
        • Numerical summaries(mean , medium, mode, variance) 数据汇总
      • Inferential statistics 推断统计
        • use sample data to make estimates, decisions, predictions, or other generalisations about a large set of data. 利用样本数据推断总体特征的统计方法

    • Data types 数据类型

      • Qualitative(or categorical) data: each data point is classified into one of a given set of categories.
        • Nominal data 分类数据
        • Ordinal data 顺序数据
      • Quantitative(or numerical data) : each data point is measured on a naturally occurring numerical scale 数据值数据
        • Height, weight, income, etc.

    • Numerical descriptive statistics

      • Measures of the center of the data
        • Mean 平均值, median 中位数, mode 众数
      • Measures of variability
        • Variance 方差，standard deviation 标准差, range 极差, interquartile range 四分位差

    • Skewed distribution

      • Symmetric: mean = median
      • Positively skewed: mean > median
      • Negatively skewed: mean < median

    • The empirical rule 经验法则（前提：对称分布）

      • 68% of the measurements will fall within 1 standard deviation of the mean
      • 95% of the measurements will fall within 2 standard deviation of the mean
      • 99.7% of the measurements will fall within 3 standard deviation of the mean
      • 该规则用于识别outliers（离群点），

    • Probability 概率

      • Basic definitions

        • P(A) >=0 && P(A) <= 1
        • P(～A) =1-P(A)， P(~A)也可以表示为P(A’)
        • Proabilities can be objective or subjective

      • Combining probablilities

        • P(AUB)=P(A)+P(B)–P(A∩B),若A与B互斥(Mutually exclusive)，则P(AUB)=P(A)+P(B)

      • Conditional probailities

        • P(A | B) = P(A ∩ B) / P(B) , P(B) > 0

      • Independent events

        • 两个事件独立的条件：P (A | B) = P(A | ~B) = P(A), and P(B | A) = P(B | ~A) = P(B)，一个事件发生与否不影响另一个事件发生的概率
        • 如果事件A，B相互独立：P(A ∩ B) = P(A | B) P(B) = P(A) P(B)

      • Bayes’ Theorem （贝叶斯公式，逆概率公式）

        • P(A |B)=P(B|Ai)P(Ai) / P(B|A )P(A )+…+P(B|A )P(A)
        • P(A) is prior proabilities为原因A的先验概率，P(A|B) 后验概率

      • Random variables

        • Sample space 样本容量
        • Random variable 随机变量
          • 离散型随机变量(discrete)
            • We must have p(x)≥0 for all x,and ∑p(x) = 1
            • Mean(or expected value):μ=∑x*p(x).
            • Variance: σ^2 = ∑(x – μ)^2 p(x)
          • 连续型随机变量(continuous)
            • P(a ≤ x ≤ b) = ∫f(x)dx
            • f(x) >= 0 for all x and ∫f(x)dx = 1
            • Mean: μ = ∫x f(x)dx
            • Variance: σ = ∫(x − μ) f(x)dx

      • The binomial distribution 二项分布

        • 成功概率为P，失败概率为1-P

        • P(x) =( n! / x!(n-x)! ) * p^x(1-p)^x, for 0<= x <= 1

        • For x ~ Bin(n,p)
          Mean of x: μ = np. Variance of x: σ^2 = np(1-p)

      • The uniform distribution 正态分布

        • if x~Uniform(c,d), when c <= x <= d, f(x) = 1/(d-c), otherwise f(x) is 0
        • Mean: μ = (c + d) / 2
        • Variance: σ^2 = (d – c)^2 / 12

      • Computing Z-scores

        • If x is distributed according to N(μ, σ), then z = (x − μ) / σ will be distributed according to the

          standard normal distribution, N(0,1)

      • Using $\bar{x}$ to estimate μ

        • If the population is normally distributed with mean μ and standard deviation σ, then the sample mean x is also normally distributed, with mean μ and standard deviation σ / N .
        • Central Limit Theorem 中心极限理论
          • If the population has any distribution with mean μ and standard deviation σ, and if N ≥ 30, then the sample mean x is normally distributed, with mean μ and standard deviation σ / N .
