Leetcode 20 Valid Parentheses(有效的括号)

Posted on 2020-01-03 Edited on 2020-03-05 In leetcode Views:

Difficulty : Easy

Type: String && Stack

Language : Java

  1. Clarification
    Given a string containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid.

    An input string is valid if:

    1. Open brackets must be closed by the same type of brackets.
    2. Open brackets must be closed in the correct order.

    Note that an empty string is also considered valid.

  2. Possible Solutions

    • Stack

      遍历字符串里的字符，如果遇到左括号，入栈，否则如果栈不为空，或栈顶元素不是右括号，就返回false，最终栈中若为空，说明没有左括号或者右括号剩余，说明此时是括号是两两匹配的（此题限定字符中只有括号，不含其他字符）

    • 字符串替换方式

      假设输入的字符串是:
      (([[{}]]))
      如果字符串是有效的，不管是嵌套结构，还是非嵌套结构，最里面肯定有一对括号是配对的，不是()，就是[]，或者是{}
      然后我们就将{}替换成””
      于是字符串就变成了(([[]]))
      重复这个逻辑，再不断的替换[]，将其改为””
      再将()替换为””
      最终全部替换完，得到的就是一个”” 空字符串。

      这个方法因为用到正则表达式，运行时间很慢，不推荐使用
