Leetcode 96 Unique Binary Search Trees(不同的二叉搜索树)

2020-03-28 • 分类: leetcode • 标签: Dynamic Programming, tree

Difficulty : Medium

Type: tree,Dynamic Programming

Language : Java

Clarification

Given n, how many structurally unique BST’s (binary search trees) that store values 1 … n?

Possible Solutions

  • 记忆化搜索（递归）
    通过HashMap实现记忆化搜索，把递归过程中求出的解保存起来，第二次需要的时候直接拿即可。每次求根结点的左子树数量和右子树数量，当前根的数量就是左子树数量和右子树数量的乘积。

  • 动态规划
    假设n个节点存在二叉排序树的个数是G(n)，令f(i)为以i为根的二叉搜索树的个数，则
    G(n) = f(1) + f(2) + f(3) + f(4) + … + f(n)G(n) = f(1)+f(2)+f(3)+f(4)+…+f(n)

    当i为根节点时，其左子树节点个数为i-1个，右子树节点为n-i，则
    f(i) = G(i-1)*G(n-i)f(i) = G(i−1)∗G(n−i)

    综合两个公式可以得到 卡特兰数 公式
    G(n) = G(0)G(n-1)+G(1)(n-2)+…+G(n-1)*G(0)G(n) = G(0)∗G(n−1)+G(1)∗(n−2)+…+G(n−1)∗G(0)
