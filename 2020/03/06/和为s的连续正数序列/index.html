和为s的连续正数序列

Posted on 2020-03-06 In 剑指Offer Views:

Difficulty : Easy

Type: Math, Sliding Window

Language : Java

Clarification

输入一个正整数 target ，输出所有和为 target 的连续正整数序列（至少含有两个数）。

序列内的数字由小到大排列，不同序列按照首个数字从小到大排列。

1 <= target <= 10^5

Possible Solutions

  • 滑动窗口
    第一次做滑动窗口类型的题，我一开始的思路是在考虑等差数列，但也没想出来，这题使用滑动窗口方法很好解决。

    滑动窗口的重要性质是：窗口的左边界和右边界永远只能向右移动，不能向左移动，这样保证了时间复杂度为O(N)。这道题里需要判断的就是滑动窗口的和与target之间的大小关系，如果sum小于target，就让右边界向右移动1格，窗口中多了一个数字，sum也要累加上这个数字。当sum大于target的时候，窗口的和需要减少，左边界向右移一格，sum减掉原来左边界所指的数字。当sum等于target，新建长度为右边界 - 左边界的数组并赋值，即得到一个解，加到ArrayList里。

Code

Test Cases

示例 1：

输入：target = 9
输出：[[2,3,4],[4,5]]

示例 2：

输入：target = 15
输出：[[1,2,3,4,5],[4,5,6],[7,8]]
