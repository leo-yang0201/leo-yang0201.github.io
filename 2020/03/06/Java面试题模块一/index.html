最近陆续的开始投暑期实习简历了，自己也还是没有信心能找到一份Java开发的实习，对面试很恐惧，因为自己不会的实在是太多了。趁着现在这个期中假期，多整理一些Java的面试题目来巩固下基础。

JDK和JRE有什么区别？

  • JDK : Java Development Kit的简称，是Java开发工具包，提供了Java的开发环境和运行环境
  • JRE : Java Runtime Environment的简称，是Java的运行环境

简单说，如果需要运行Java程序，只需要安装JRE就行，如果需要编写Java环境，则需要安装JDK，JDK包含了JRE及编译器javac。

private,protected,public和default的区别

  • public
    它具有最大的访问权限，可以访问任何一个在CLASSPATH下的类，接口，异常等。它往往对于对外的情况，也就是对象或类对外的一种接口的形式。
  • protected
    它主要的作用就是用来保护子类的。它的含义在于子类可以使用它修饰的成员，其他的不可以，它相当于传递给子类的一种继承的东西。
  • Default
    有的时候也称为friendly（友员），它是针对本包访问而设计的，任何处于本包下的类，接口，异常等，都可以互相访问，即使是父类没有用protected修饰的成员也可以。
  • private
    它的访问权限仅限于类的内部，是一种封装的体现，例如，大多数的成员变量都是修饰为private的，它们不希望被其他任何外部的类访问。

两个对象的hashCode()相同，则equals()也一定为true？

不对，两个对象的hashCode()相同，equals()不一定为true

两个对象的hashcode相同，equals返回的是false，因为在散列表中，hashCode相等是两个键值对的哈希值相等，但是哈希值相等，不一定键值对相等。

final在Java中的作用

  • final修饰的类不能被继承
  • final修饰的方法一旦初始化不能重写
  • final修饰的变量是常量，初始化后不能改变

Java中的Math.round(-1.5)等于多少

等于-1，因为在数轴上取值时，中间值(0.5)向右取整，所以+0.5是向上取整，-0.5是直接丢弃

String属于基础的数据类型吗？

不属于，基础类型有8种，byte,short,int,long,float,double,char,boolean，String属于对象

Java中操作字符串都有哪些类，它们之间有什么区别？

操作字符串的类有String,StringBuffer,StringBuilder

  • String声明的是不可变对象(定义为final)，每次操作都会生成新的对象，然后将指针指向新的对象，StringBuffer和StringBuilder是对原有对象基础上进行操作，所以在经常改变字符串内容的情况下最好不要用String
  • StringBuffer是线程安全的，StringBuilder是线程不安全的，StringBuilder比StringBuffer性能高，单线程推荐用StringBuilder，多线程推荐用StringBuffer

String str=”i”与 String str=new String(“i”)一样吗？

不一样，前者JVM会将其分配到常量池，后者会被分到堆内存中。常量池存放String常量和基本类型常量(public static final), 内存堆存放所有New出来的对象，内存栈存放基本类型的变量数据和对象的引用，对象本身不存在栈里而是堆里或常量池中（String常量对象）

String类常用方法都有哪些？

  • indexOf(): 返回指定字符的索引
  • charAt(): 返回指定索引处的字符
  • replace(): 字符串替换
  • trim(): 去除字符串两端空白
  • split(): 分割字符串，返回一个分割后的字符串数组
  • getBytes(): 返回字符串的byte类型数组
  • length(): 返回字符串长度
  • toLowerCase(): 将字符串转为小写字母
  • toUpperCase(): 将字符串转为大写字母
  • substring(): 截取字符串
  • equals(): 字符串比较

抽象类必须有抽象方法吗？

不需要，抽象类不一定要有抽象方法

普通类和抽象类的区别？

  • 普通类不能包含抽象方法，抽象类可以包括抽象方法
  • 抽象类不能直接实例化，普通类可以直接实例化

抽象类能用final修饰吗？

不能，因为抽象类必须被子类继承后才能创建实例化对象，定义为final后便不能被继承，产生矛盾

接口和抽象类有什么区别

  • 实现：抽象类的子类使用extends继承，接口必须使用implements来实现接口
  • 构造函数：抽象类可以有构造函数，接口不能有
  • main方法：抽象类可以有main方法且能运行，接口不能有
  • 实现数量：类可以实现很多接口，但只能继承一个抽象类
  • 访问修饰符：接口方法默认以public修饰，抽象类没有限制

Java中I/O留分为几种

按功能分：输入流(input)和输出流(output)

按类型分：字节流和字符流

字节流按8位传输，以字节为单位输入输出数据，字符流按16位传输以字符为单位输入输出数据。

BIO，NIO，AIO有什么区别？

  • BIO：Block IO 同步阻塞式IO，就是平常使用的传统IO，特点是模式简单使用方便，并发处理能力低
  • NIO：New IO 同步非阻塞IO，是传统IO的升级，客户端和服务器通过Channel(通道)通讯，实现了多路复用
  • AIO：Asynchronous IO是NIO的升级，也叫NIO2，实现异步非堵塞IO，异步IO的操作基于事件和回调机制

Files的常用方法有哪些？

  • Files.exists(): 检查文件路径是否存在
  • Files.createFile(): 创建文件
  • Files.createDirectory(): 创建文件夹
  • Files.delete(): 删除一个文件或目录
  • Files.copy(): 复制文件
  • Files.move(): 移动文件
  • Files.size(): 查看文件个数
  • Files.read(): 读取文件
  • Files.write(): 写入文件
