Leetcode 100 Same Tree(相同的树)

2020-03-27 • 分类: leetcode • 标签: BFS, DFS, tree

Difficulty : Easy

Type: tree,DFS,BFS

Language : Java

Clarification

Given two binary trees, write a function to check if they are the same or not.

Two binary trees are considered the same if they are structurally identical and the nodes have the same value.

Possible Solutions

  • DFS
    深度优先搜索，提前构造好递归的结束条件，当结点都为空的时候返回true，只有一个为空或者结点值不同返回false，递归访问左孩子和右孩子，&&运算计算两边是否都为true
  • BFS
    广度优先搜索，两个Queue实现。
