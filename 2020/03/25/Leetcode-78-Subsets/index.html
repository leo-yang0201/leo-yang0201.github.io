Leetcode 78 Subsets(子集)

2020-03-25 • 分类: leetcode • 标签: Array, Backtracking, Bit Manipulation

Difficulty : Medium

Type: Array,Backtracking,Bit Manipulation

Language : Java

Clarification

Given a set of distinct integers, nums, return all possible subsets (the power set).

Note: The solution set must not contain duplicate subsets.

Possible Solutions

  • 回溯算法

    添加一个数，递归，删除之前的数，下次循环

  • 位运算（异或）

    数组的每个元素，可以有两个状态，在子数组中和不在子数组中，所有状态的组合就是所有子数组了。

    例如，nums = [ 1, 2 , 3 ]。1 代表在，0 代表不在。
