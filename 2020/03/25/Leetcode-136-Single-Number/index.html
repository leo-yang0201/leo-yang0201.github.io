Leetcode 136 Single Number(只出现一次的数字)

Posted on 2020-03-25 In leetcode Views:

Difficulty : Easy

Type: Hash Table,Bit Manipulation

Language : Java

Clarification

Given a non-empty array of integers, every element appears twice except for one. Find that single one.

Note:

Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Possible Solutions

  • 哈希表

    新建一个Hashmap，两次循环，第一次循环把nums数组里的值作为key，出现次数count作为value依次put进去，第二次循环map的key列表，看哪个key的count是1，返回那个key。

  • 位运算（异或）

    题目要求不用额外空间实现，所以哈希表不是最佳解法，最好的方法是通过位运算里的异或运算。

    异或运算特点：

    • 一个数和0做异或运算等于本身
    • 一个数和其本身做异或运算等于0
    • 异或运算满足交换律和结合律

    将所有数字按顺序做异或运算，由于交换律和结合律，只要出现过两次的数字最后都会变成自己和自己做异或运算，最后就是0和那个只出现一次的数字做异或运算，所以循环后得到的最后计算结果就是只出现一次的数字。
