Leetcode 350 Intersection of Two Arrays II(两个数组的交集II)

Difficulty : Easy

Type: Hash Table,Two Pointers,Binary Search,Sort

Language : Java

Clarification

Given two arrays, write a function to compute their intersection.

Note:

  • Each element in the result should appear as many times as it shows in both arrays.
  • The result can be in any order.

Follow up:

  • What if the given array is already sorted? How would you optimize your algorithm?
  • What if nums1‘s size is small compared to nums2‘s size? Which algorithm is better?
  • What if elements of nums2 are stored on disk, and the memory is limited such that you cannot load all elements into the memory at once?

Possible Solutions

  • 哈希表
    HashMap进行映射，先用HashMap记录第一个数组中的元素到key里，每个元素出现的次数放在value里。然后遍历第二个数组，若找到对应元素且该元素出现的次数大于0，就添加这个元素到list里，并且HashMap的value - 1，表示已经找到一个相同的了。最后把list的值遍历取出赋值给数组。
  • 双指针
    先对两个数组进行排序，然后设双指针分别指向第一个数组和第二个数组的头部，如果nums[p1] == nums[p2]，说明两个数组都有这个数，加入list中。如果不等，移动小的那个指针。
