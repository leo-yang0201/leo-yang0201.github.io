Leetcode 72 Edit Distance(编辑距离)

Difficulty : Hard

Type: String,Dynamic Programming

Language : Java

Clarification

  • Given two words word1 and word2, find the minimum number of operations required to convert word1 to word2.

    You have the following 3 operations permitted on a word:

    1. Insert a character
    2. Delete a character
    3. Replace a character

Possible Solutions

  • 动态规划
    这道题我一开始看了题解也没有想明白，但是这种题面试中被问到的频率还挺高的，动态规划一类的题自己也还是比较怕遇到。
    这道题需求求解的是word1编辑到word2需要的步骤最小值，思路是可以先把word1的一个子串编辑成word2的一个子串，然后随着子串的长度逐渐变大，推导出结果。
    用w1si表示word1的子串(sub(0,i))，w2sj表示word2的子串(sub(0,j));
    二维数组dp[i][j]表示把w1si编辑成w2sj所需要的最少步数
    假如现在word1是horse，word2是ros
    把horse转为Ros，可在已知以下三种情况之下，在做一个额外操作：

    • 当前已经有hors编辑为ros的步骤，那么可以在原word1(horse)的基础上，删除最后的e，实现编辑，即 horse -> rose -> ros
    • 当前已经有horse编辑成ro的步骤，那么可以在原word1(horse)的基础上，插入一个s，也可以得到ros，即horse -> ro -> ros
    • 当前已经有了hors编辑成ro的步骤，那么可以在原word1(horse)的基础上，把最后的一个e替换成s，也可以得到ros，即 horse -> roe -> ros

    注：hors -> ros 等于 hor -> ro

    状态转移方程

    可以认为上面三种情况下最小值就是最终结果：

    dp(horse -> ros) = min{dp(hors->ros),dp(horse->ro),dp(hors->ro)} + 1

    即：dp[i][j] = min(dp[i - 1][j],dp[i][j - 1],dp[i - 1][j - 1] + 1);

    特殊情况下：

    如果word1[i] == word2[j]，那么dp[i][j]=dp[i - 1][j - 1];

    边界

    边界是i = 0 或者 j = 0

    如果i = 0，表示从horse的一个空子串(“”)编辑成ros的所有子串(“”、”r”、”ro”、”ros”)所需要的步数，每一个都执行插入就可以了，结果为dp[0][j] = j;

    如果j = 0，表示从horse的所有子串(“”、”h”、”ho”、”hor”、”hors”、”horse”)编辑成ros的一个空子串(“”)所需要的步数，每一步都执行删除就可以了，结果为dp[i][0] = i;
