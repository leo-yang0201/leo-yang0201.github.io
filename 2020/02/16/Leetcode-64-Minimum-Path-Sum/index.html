Leetcode 64 Minimum Path Sum(最小路径和)

发表于 2020-02-16

Difficulty : Medium

Type: Dynamic Programming

Language : Java

  1. ClarificationGiven a m x n grid filled with non-negative numbers, find a path from top left to bottom right which minimizes the sum of all numbers along its path.

    Note: You can only move either down or right at any point in time.

  1. Possible Solutions

    • 动态规划
      题目要求，只能向右或者向下走，意味着当前单元格(i,j)只能从左单元格(i-1,j)和上单元格(i,j-1)走到，因此只需要考虑矩阵左边界和上边界
      • 走到当前单元格(i,j)的最小路径和 = 从左方单元格(i-1,j)与从上方单元格(i,j-1)走来的两个最小路径和中较小的+当前单元格值，具体有4种情况
        • 左边和上边都不是矩阵边界(i!=0,j!=0),dp[i][j] = min(dp[i-1][j],dp[i][j-1]) + dp[i][j]
        • 只有左边是边界时，只能从上边来,dp[i][j] = dp[i][j-1] + dp[i][j
        • 只有上边是边界时，只能从左边来,dp[i][j] = dp[i-1][j] + dp[i][j]
        • 左边和上边都是边界，就是起点，dp[i][j] = grid[i][j]
