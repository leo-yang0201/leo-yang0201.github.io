Leetcode 92 Reverse Linked List II(反转链表2)

发表于 2020-02-13

Difficulty : Medium

Type: Linked List

Language : Java

  1. Clarification

    Reverse a linked list from position m to n. Do it in one-pass.

    Note: 1 ≤ m ≤ n ≤ length of list.

  2. Possible Solutions

    第一步：找到待反转节点的前一个节点。
    第二步：反转m到n这部分。
    第三步：将反转的起点的next指向反转的后面一部分。
    第四步：将第一步找到的节点指向反转以后的头节点。
