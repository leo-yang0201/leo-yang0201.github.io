Leetcode 230 Kth Smallest Element in a BST(二叉搜索树中第K小的元素)

Posted on 2020-02-10 Edited on 2020-03-08 In leetcode Views:

Difficulty : Medium

Type: Tree, Binary Search

Language : Java

  1. Clarification

    Given a binary search tree, write a function kthSmallest to find the kth smallest element in it.

    Note:
    You may assume k is always valid, 1 ≤ k ≤ BST’s total elements.

  2. Possible Solutions

    • 中序遍历法
      二叉搜索树BST有一个重要性质：中序遍历为排序数组（其所有结点的val按升序排列），这样根据中序遍历，每找到一个元素判断其是否为第k个元素即可
